# ClassyShop - Big Mega Store

## Tech Stack
- **Frontend**: React 18, React Router 6, Axios, React Toastify, React Icons
- **Backend**: Node.js, Express, MongoDB, Mongoose, JWT, bcryptjs

## Setup & Run

### Prerequisites
- Node.js installed
- MongoDB running locally on port 27017

### Backend
```bash
cd backend
npm install
npm run seed     # Seed database with products & categories
npm run dev      # Start backend (port 5000)
```

### Frontend
```bash
cd frontend
npm install
npm start        # Start frontend (port 3000)
```

### Open browser
Go to: http://localhost:3000

## Features
- Hero slider with 3 slides (auto-play)
- 8 categories: Fashion, Electronics, Bags, Footwear, Groceries, Beauty, Wellness, Jewellery
- Popular Products tabs by category
- Latest & Featured Products sections
- Promotional banners & deal sections
- Blog section
- Full auth (Register/Login/Profile)
- Shopping Cart (localStorage)
- Wishlist (requires login)
- Checkout with Pakistani payment methods
- Order tracking
- Product reviews
- Search functionality
- Pakistani Rupee (Rs.) currency
